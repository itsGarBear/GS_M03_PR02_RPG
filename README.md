# GS_M03_PR02_RPG

<b>Left Click</b> to attack / place block

<b>Right Click</b> to cycle through inventory

    - Ice/Water blocks have colliders and act as walls
    - Other blocks act as a ground that can be walked on
    - You can place blocks on top of each other

Link to playable: https://itsgarbear.github.io/GS_M03_PR02_RPG/
